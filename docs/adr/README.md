# Architecture Decision Records

ADRs document significant technical and UX decisions. Create an ADR **before**
implementing when a plan’s “Decisions & ADRs” section lists one.

## Naming

`NNN-short-kebab-title.md` — e.g. `001-tech-stack.md`

## Template

Use [`template.md`](./template.md).
